# INFO833 DHT  

## Presentation  
The goal of this project is to build a DHT. 

## How install  
```
git clone https://github.com/ForrayGabriel/INFO833-PeerSim-DHT 
``` 

Make sure you have the following packages :  
``` 
djep-1.0.0.jar
jep-2.3.0.jar
```   
Then run the peersim.Simulator.java by adding : 
```
config_file.cfg
```
